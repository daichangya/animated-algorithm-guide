# 关于我们

## 关于本书

《一看就懂的算法动画》是我们的重点项目之一，这是一本通过动画演示帮助读者直观理解算法的书籍。本书是 [ZThinker](https://algo.jsdiff.com/) 平台的算法学习系列内容的一部分。

### 本书特点

- 通过动画演示算法执行过程
- 详细讲解算法原理
- 提供多种编程语言实现
- 包含练习题和思考题
